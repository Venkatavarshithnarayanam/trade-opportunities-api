HOW IT WORKS NOW
================================================================================

1. Try to start server on port 8000
2. If OSError (address in use) → Catch it
3. Increment port to 8001
4. Try again
5. Repeat until success or all ports exhausted

No pre-checks. No false positives. Just real exceptions and retries.
