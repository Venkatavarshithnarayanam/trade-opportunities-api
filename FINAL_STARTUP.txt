TROUBLESHOOTING
================================================================================

Q: API won't start
A: Check logs for errors, ensure psutil is installed

Q: "Address already in use" error
A: API automatically retries on next port (8001, 8002, etc.)

Q: Validation script fails
A: Check which port API is running on in the logs

Q: "Access Denied" error
A: This is handled automatically - API uses next available port

Q: Port 8000 still in use after termination
A: API automatically uses port 8001, 8002, etc.
